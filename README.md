# Campus Task Manager

A simple **command-line task manager** for students to keep track of assignments, exams, projects, and personal tasks. Tasks can be added, viewed, edited, marked as completed, and deleted. The application also highlights tasks that are due soon.

---

## Features

- Add new tasks with title, category, and deadline
- View all tasks with status
- Highlight tasks due within 2 days
- Mark tasks as completed
- Edit existing tasks (title or deadline)
- Delete tasks
- Tasks are stored in a JSON file (`tasks.json`) for persistence

---

## Requirements

- Python 3.11+  
- Standard Python libraries: `json`, `os`, `datetime` (no external dependencies)
