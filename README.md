# Campus Task Manager

Campus Task Manager is a simple and efficient Python application designed to help students keep track of academic tasks, personal reminders, deadlines, and campus activities—all in one place.
It uses a clean console-based interface and stores all data permanently in a JSON file.

The tool highlights upcoming deadlines, allows task editing, status updates, and deletion, making student life more organized and stress-free.

# Features
1.Add Tasks Easily
Each task includes:
Title
Category
Deadline (YYYY-MM-DD)
Status (Pending/Completed)
2.View All Tasks
Displays all tasks in a clear, readable list.
3.Deadline Alerts
Shows tasks that are due within 2 days, helping you stay ahead.
4.Update & Edit Tasks
You can modify:
Title
Deadline
Status
5.Delete Completed or Unwanted Tasks
Clean up old tasks anytime.
6.Safe & Lightweight
Uses only Python’s built-in modules:
json, os, datetime

# How to Run the Program
Install Python 3.x
Download or clone the project folder
Open the folder in Terminal / CMD
Run the program:
python main.py

# How to Use
1. Add Task
Enter:
Task title
Category (Assignment/Exam/Project/Personal)
Deadline
2. View All Tasks
Displays a list with:
Task number
Title
Category
Deadline
Status
3. View Tasks Due Soon
Shows tasks that are due today or within 2 days.
4. Mark a Task Completed
Select task number → status changes to "Completed".
5. Edit a Task
Modify:
Title
Deadline
(Leave fields blank to keep old value)
6. Delete a Task
Remove task permanently by its number.
7. Exit
Safely closes the program.
